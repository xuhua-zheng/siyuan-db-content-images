# 数据库内容图

把数据库行绑定的笔记/块中插入的图片，同步到数据库的资源字段中。

## 使用方法

1. 在目标数据库中新建一个资源字段，默认命名为 `内容图`。
2. 把笔记或块加入这个数据库。
3. 右键数据库项目，执行 `插件 > 同步内容图`；也可以打开命令面板，执行 `同步当前数据库内容图`。
4. 如果想换字段名，进入插件设置修改 `资源字段名`。
5. 如果希望在添加数据库行后自动同步，打开 `自动同步可见数据库`。

插件会读取每一行主键绑定的块，提取其中的 Markdown 图片、HTML 图片，以及索引中的资源图片，并写入配置的资源字段。

也可以在文档编辑区使用右键菜单：

- 右键某张图片，执行 `插件 > 追加到内容图`，把这张图片追加到匹配数据库项目的 `内容图` 字段中。
- 在正文/块内容中右键，执行 `插件 > 从内容图插入图片`，把当前块或当前文档绑定数据库里的 `内容图` 图片作为独立图片块插入到所选块后方；如果存在多个可用数据库/字段来源，会先弹窗选择。

这两个入口都会检查图片所在块和当前文档绑定的数据库。

同步和插入都会使用已有图片资源的路径/链接，不会复制或新建图片附件。

建议把该资源字段作为专用字段使用。默认会覆盖该字段现有值，让它始终反映笔记正文中的图片；如果不希望覆盖，关闭 `覆盖已有资源值`。

## 限制

插件不能新增思源内核级数据库列设置，所以它不会在字段菜单里增加原生开关。它使用公开 API 同步一个普通资源字段，适合不改内核、可打包分发的场景。
